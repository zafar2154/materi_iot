CSS Font

font-family : untuk mengatur jenis font yang akan digunakan

font-size

font-weight untuk mengatur ketebalan

Font variant untuk mengubah font menjadi small caps

font-style: italic

font-style: oblique

line-height: untuk mengatur spasi antar baris

CSS Text Styling

color: mengatur warna pada teks. Warna bisa berupa teks yg disediakan autocomplete vscode, rgb([red], [green], [blue]), code hex (contoh : #000fff), dan hsl(39,100%,50%)

text-align = Mengatur format paragraf/teks
Value : center, justify, left, right

text-decoration = mengatur dekorasi pada teks.
value : line-through overline underline

text-transform = Mengubah jenis huruf menjadi kapital atau besar HURUF KECIL

word-spacing : mengatur jarak antar kata

letter-spacing = mengatur jarak antar huruf

Background

Kita dapat mengatur background pada suatu elemen. Pada body web ini menggunakan property background-image. Dan pada elemen ini menggunakan background-color

display digunakan untuk menentukan bagaimana suatu elemen ditampilkan pada halaman web. display memiliki value : inline, block, inline-block, dan none

Float

adalah properti CSS yang digunakan untuk mengatur bagaimana elemen mengalir dalam sebuah halaman web, terutama dalam mengatur tata letak teks dan gambar. Properti ini memungkinkan elemen untuk "mengapung" di sebelah elemen lain, sehingga konten lain dapat mengalir di sekitarnya.
